# nakla.fun - The Website

This repository contains sources for [nakla.fun](nakla.fun) website.

# Idea behind it

Feel free to commit ;)

# Installation and usage

Bootstrapping:

```bash
npm install
node master --help
```

Build

```bash
node master build
```

Now open `/docs`- it contains your webpages. Easy, isn't it? =)

> Why `/docs`? We're serving through GH Pages.
